# 🛡️ Safe Alert – The Recall Tracking System

**Role:** Data Engineer, Data Visualization Specialist  
**Team Size:** 5  
**Timeline:** Spring 2025  
**Tools:** Python, openFDA API, Google Colab, gspread, Google Sheets, Tableau, HTML, CSS, Firebase  

## 🚀 Overview
Safe Alert is a web-based recall tracking system designed to help families, restaurants, and grocery professionals stay informed about food safety. It aggregates real-time recall data from the FDA and delivers notifications to users via dashboards and filtering tools.

## 🔧 My Contributions
- Built Python pipeline using openFDA API + gspread to extract and clean 1,000+ records.
- Automated upload of structured data to Google Sheets for live tracking.
- Visualized food recall trends using Tableau dashboards.
- Filtered recalls by category (meat, dairy, seafood, etc.) to enhance usability.

## 🧪 Features
- Real-time FDA food recall integration
- Category filters for targeted searches
- Firebase-based newsletter signup (team built)
- PayPal donation system (planned)

## 📊 Survey Results
- 80%+ users unaware of FDA recall tools
- 90% expressed interest in receiving recall alerts
- Demand for cleaner, more accessible tracking platform
